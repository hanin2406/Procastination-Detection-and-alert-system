# 📊 Student Procrastination Detection & Productivity Enhancement System

## 📌 Overview

The **Student Procrastination Detection & Productivity Enhancement System** is a mini project developed to analyze student procrastination behavior using **Machine Learning** and reduce distractions through a **browser extension**.

The system predicts whether a student is likely to procrastinate based on behavioral factors such as **study hours, phone usage, and sleep duration**. It also provides **productivity insights, recommendations, and real-time distraction alerts** through a browser extension.

---

## 🎯 Objectives

- Analyze student behavior patterns
- Predict procrastination using Machine Learning
- Provide productivity insights and recommendations
- Monitor distractions through a browser extension
- Improve student focus and time management

## ⚙️ How It Works

1. User data is collected through **Google Forms**
2. Dataset is stored in **CSV format**
3. Data is preprocessed and cleaned
4. Important features are selected:
   - Study Hours
   - Phone Usage
   - Sleep Duration
5. Logistic Regression model predicts procrastination
6. Streamlit web app displays:
   - Prediction result
   - Productivity score
   - Behavioral profile
   - Recommendations
7. Browser extension monitors distracting websites and provides alerts
